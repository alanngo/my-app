A(ng)ular Commands

  • (n)ew

    - creates a new angular workspace

    - takes a while to do it

  • (g)enerate

    - creates/modifies files based on schematic

    - Common things to generate...

    - (c)omponent
    - (s)ervice
    - (d)irective
    - (p)ipe

  • (s)erve

    - Runs the angular app

    - Reads from app-component.html first

    - After you stop the app, run "$ pkill -9 node"

More commands can be found by running "$ ng help"
